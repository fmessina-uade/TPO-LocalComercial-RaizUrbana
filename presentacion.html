3. Mejoras y Funcionalidades (Segunda Entrega)

Esta sección detalla la transformación del sitio estático a una plataforma interactiva, gestionada principalmente por JavaScript (promociones.js).

Lógica de Negocio y Calculadora Interactiva

  • Modal de Selección de Productos: Se reemplazó un sistema de selección básico por un modal interactivo (imágenes, nombres y precios).
  • Contador de Cantidades: Se añadió un sistema de contador para seleccionar múltiples unidades del mismo artículo.
  • Límite de Selección (Regla de Negocio): Restricción que limita la selección a un máximo de 3 unidades totales en el carrito.
  • Promociones Condicionales (Lógica 3x2):
    • Productos específicos designados como "aptos para 3x2" con un distintivo visual.
    • La promoción "3x2" se activa/desactiva automáticamente en el desplegable según los productos elegidos.

Mejoras de Interfaz y Experiencia de Usuario (UI/UX)

  • Botón Condicional: El botón "Calcular" permanece desactivado (disabled) hasta que se selecciona al menos un producto.
  • Spinner de Carga: Indicador de carga (spinner) en el botón "Calcular" para dar respuesta visual.
  • Diseño Cohesivo: Se aplicaron los estilos de las tarjetas (values-card) de "Nosotros" a las promociones.

4. Estrategia: Bootstrap + Tailwind juntos

Decisiones

  • Bootstrap 5: Usado para grid, espaciados, tipografías y componentes base (cards, badges).
  • Tailwind (Play CDN): Usado para utilidades puntuales (con prefijo tw-) sin tocar el CSS global.

Configuración

Se desactiva preflight de Tailwind y se define prefix: 'tw-' para evitar conflictos.

<script>
  tailwind = {
    config: {
      corePlugins: { preflight: false },
      prefix: 'tw-',
      theme: {
        extend: {
          colors: {
            brand: '#2E7D32',
            brandDark:'...'
          }
        }
      }
    }
  }
</script>
<script src="https://cdn.tailwindcss.com"></script>
